# Инструкция по админке «МираВкус»

Этот документ сделан для менеджера без технической подготовки.  
Работайте строго по шагам ниже.

---

## 1) Вход в админку

1. Откройте сайт: `https://miravkus.ru/admin`
2. Введите e-mail и пароль администратора.
3. Нажмите **«Войти»**.

Если забыли пароль:
- нажмите **«Забыли пароль?»**
- введите e-mail
- нажмите **«Отправить письмо для сброса»**

---

## 2) Главное правило безопасности

- После входа сразу поменяйте пароль через кнопку **«Сменить пароль»** (вверху админки).
- Не передавайте пароль в мессенджерах без необходимости.
- Если случайно сделали не то — не паникуйте, почти все можно исправить повторным сохранением.

---

## 4) Быстрые сценарии (самое нужное)

## 4.1 Добавить товар

1. Откройте **«Товары»**.
2. Нажмите **«Добавить товар»**.
3. Заполните обязательные поля:
   - название
   - цена
   - краткое описание
   - категория
4. Добавьте фото (или вставьте ссылку на фото).
5. Нажмите **«Добавить»**.

Чтобы изменить товар:
1. Нажмите на карточку товара в списке.
2. Внесите правки.
3. Нажмите **«Сохранить»**.

Чтобы удалить:
1. Нажмите иконку удаления у товара.
2. Подтвердите действие.

---

## 4.2 Добавить категорию

1. Откройте **«Категории»**.
2. Нажмите **«Добавить категорию»**.
3. Заполните:
   - **ID** (латиницей, например `candy`)
   - название (как увидит клиент)
   - emoji и цвет
4. Нажмите **«Добавить»**.

Важно:
- ID категории лучше не менять после начала продаж.
- Для главной страницы включайте показ категории и порядок показа.

---

## 4.3 Добавить упаковку

1. Откройте **«Упаковка»**.
2. Нажмите **«Добавить упаковку»**.
3. Заполните:
   - ID (например `gift_wrap`)
   - название
   - цену
4. Нажмите **«Добавить»**.

---

## 4.4 Добавить статью

1. Откройте **«Статьи»**.
2. Нажмите **«Добавить статью»**.
3. Заполните:
   - заголовок
   - краткое описание карточки
   - полный текст
4. Добавьте фото.
5. Нажмите **«Добавить»**.

Совет:
- Пишите короткие абзацы.
- Проверяйте, как статья выглядит на сайте сразу после сохранения.

---

## 4.5 Создать промокод

1. Откройте **«Промокоды»**.
2. Нажмите **«Добавить промокод»**.
3. Заполните:
   - код (например `SALE10`)
   - процент скидки
   - область действия (все товары / категории / конкретные товары)
4. Убедитесь, что промокод активен.
5. Нажмите **«Добавить»**.

---

## 4.6 Поменять верхнее меню (Хедер)

1. Откройте **«Хедер»**.
2. Можно менять:
   - текст бренда
   - цвет текста бренда
   - цвет кнопки «Меню»
   - цвет текста кнопки «Меню»
3. В блоке **«Предпросмотр»** сразу видно результат.
4. Нажмите **«Сохранить»**.

Важно:
- Реальный логотип не редактируется в админке и должен оставаться без изменений.

---

## 4.7 Поменять блок «О нас»

1. Откройте **«О нас»**.
2. Измените заголовок, подзаголовок, текст.
3. При необходимости добавьте/замените фото.
4. Нажмите **«Сохранить»**.

---

## 4.8 Поменять футер

1. Откройте **«Футер»**.
2. Обновите контакты, доставку, соцсети, копирайт.
3. Нажмите **«Сохранить»**.

---

## 4.9 Обработать заказ

1. Откройте **«Заказы»**.
2. Найдите нужный заказ.
3. Поменяйте статус (например «В обработке» → «Передан» → «Доставлен»).
4. Нажмите **«Сохранить»**.

---

## 5) Раздел «Импорт» (категории JSON)

Используется для массового добавления категорий.

Минимальный пример JSON:

```json
[
  {
    "id": "candy",
    "name": "Сладости",
    "emoji": "🍬",
    "color": "candy-pink"
  }
]
```

Если после импорта получили ошибку:
- проверьте, что это именно массив `[...]`
- у каждого объекта есть `id` и `name`

---

## 6) Проверка после любых изменений

После каждого сохранения:
1. Откройте главную страницу сайта.
2. Проверьте, что изменения видны.
3. Если не видно — обновите страницу сочетанием **Ctrl + F5**.

Если все равно не видно:
- откройте сайт в режиме инкогнито
- проверьте через 1–2 минуты (иногда задержка кэша)

---

## 7) Если сделали что-то не так

- В верхней части админки есть кнопка **«Сбросить»** — используйте только если понимаете последствия.
- В большинстве случаев проще открыть нужный раздел и сохранить правильные значения повторно.
- При критичной ошибке сразу сообщите разработчику, что именно меняли и в каком разделе.

---

## 8) Мини-чеклист для менеджера

Перед завершением работы в админке:
- Проверил(а) изменения на самом сайте
- Проверил(а) мобильную версию
- Проверил(а) контакты и цены
- Вышел(ла) из админки/закрыл(а) вкладку

Готово.
